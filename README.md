# Multi-Otsu Optimization via Micro Self-Adaptive Differential Evolution for Pneumonia Segmentation and Classification in X-Rays 🫁🧬

This project implements an advanced Computer Vision and Machine Learning pipeline for pneumonia detection in chest X-ray images. It utilizes the **Multi-Otsu** thresholding algorithm, optimized through the **Micro Self-Adaptive Differential Evolution ($\mu$SADE)** metaheuristic, to accurately isolate the Region of Interest (ROI). This is followed by radiomic feature extraction and classification using Support Vector Machines (SVM).

## 🚀 Pipeline Architecture

The system is strictly divided into two main phases to guarantee data integrity and prevent Data Leakage.

### Phase 1: Extraction & Dataset Generation (Generator)
1. **Multi-Otsu Segmentation:** The radiological image histogram is divided into $D+1$ classes (defaulting to 4 classes, 3 thresholds).
2. **$\mu$DE Optimization:** To bypass the combinatorial explosion of the traditional brute-force Otsu algorithm ($O(L^k)$), an evolutionary algorithm with micro-populations ($NP=8$) and self-adaptive parameters ($F$, $CR$) is deployed. The pipeline evaluates 4 mutation schemes:
   - `DE/rand/1`
   - `DE/best/1`
   - `DE/rand/2`
   - `DE/best/2`
3. **Radiomic Feature Extraction:** Based on the binary mask generated by the uppermost threshold (which isolates pulmonary opacities), the following features are extracted:
   - **Morphological:** Number of regions, maximum area, total area.
   - **Texture (GLCM):** Contrast, Energy, Homogeneity.
4. **Export:** All extracted features (including global illumination parameters) are saved into pure `.csv` datasets. Additionally, visual `.png` compositions of the algorithmic convergence and final segmentations are generated.

### Phase 2: Classification & Evaluation (SVM)
1. **Bias Purge (Ablation Study):** Before training, the script explicitly removes variables dependent on the global brightness of the X-ray ($P_0, P_1, P_2, \mu_2, \sigma_2$) from the dataset. This crucial step forces the model to learn from the actual morphology and texture of the lung tissue, rather than taking shortcuts based on hardware calibration artifacts.
2. **SVM Training:** An `SVC(kernel='linear')` classifier is trained on the purged datasets.
3. **Cross-Validation:** `StratifiedShuffleSplit` (10 folds, 85% Train / 15% Test) is applied alongside parallel processing via `joblib` to simultaneously evaluate the datasets generated by the different evolutionary mutation schemes.
4. **Metrics:** The model evaluates Accuracy, Recall, F1-Score, and AUC, consolidating the results into a final comparative table ranked by predictive accuracy.

## 🧠 Theoretical Foundations

This project merges three complex domains of applied computational science in the medical field:

### 1. Multi-Otsu Thresholding
The classic Otsu method is a deterministic thresholding algorithm that seeks to maximize the inter-class variance ($\sigma_B^2$) of a histogram to separate the foreground from the background. When extended to multiple thresholds (Multi-Otsu) to segment different types of lung tissue (bone, air, healthy tissue, opacities), the computational complexity grows exponentially to $O(L^k)$, where $L$ represents intensity levels and $k$ the number of thresholds. Solving this via brute force across thousands of X-rays is computationally unfeasible.

### 2. Micro-SADE ($\mu$SADE)
To resolve the Multi-Otsu bottleneck, we implemented **Self-Adaptive Micro Differential Evolution**. 
* **Micro ($\mu$):** Utilizes a tiny population size (e.g., $NP=8$). This enables extremely fast iterations and low memory consumption, which is ideal for processing thousands of medical images.
* **Self-Adaptive (SA):** Instead of manually tuning the mutation factor ($F$) and crossover rate ($CR$), the algorithm encodes them within the individual. The parameters evolve dynamically alongside the solution, automatically adapting to the topology of each specific X-ray histogram to prevent premature convergence.

### 3. Radiomics and GLCM Analysis
Radiomics is the process of extracting a large number of quantitative features from medical images. Once $\mu$SADE finds the optimal thresholds, we isolate the pulmonary region of interest. To mathematically describe this region (and differentiate pneumonia from a healthy lung), we apply the **Gray-Level Co-occurrence Matrix (GLCM)**. This technique measures the spatial texture of the image by calculating how often pairs of pixels with specific values and in a specific spatial relationship occur, yielding key descriptors like Energy (uniformity) and Homogeneity.

## ⚙️ Installation & Execution
1. Clone the repository.
2. Ensure you have the required dependencies installed: `pip install numpy opencv-python scikit-image scikit-learn pandas joblib matplotlib`
3. Place your X-ray dataset in the designated folder (e.g., `/data/xrays/`).
4. Execute the Phase 1 pipeline to generate the segmentations and extract the CSV features.
5. Execute Phase 2 to train the SVM model and visualize the comparative metrics of the evolutionary schemes.
